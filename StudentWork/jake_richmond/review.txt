1. Variable:
-   A variable is a name for something that can change, depending on conditions or information passed to the program.
A place to put something.
-   A variable is similar to a cookie in a jar
-   You would use it when you need to describe something without knowing what that thing is.

2. Operator:
-   An operator is a symbol that usually represents an action or process.
-   An example would be an "+" sign.
-   You would use an operator when you needed to return a value.

3. Assignment Operator:
-   An Assignment Operator is the operator used to assign a new value to a variable, property, event or indexer element.
-   It is similar to making x = y
-   You would use it when you need to return a specific operator

4. String:
-   A string is usually a bit of text or information that you want to display to someone, or "export" out of the program
you are writing. It is usually surrounded by double or single quotes.
